# Test Protocol

## Goal

Observe return behavior.

## Cohort

10 users maximum.

## Duration

3-7 days.

## Instructions

1. Open board.
2. Create 1-3 cards.
3. Write short notes.
4. Fill Re-entry Notes:
   - Why still open?
   - If I return, start here
5. Close.
6. Return naturally.
7. Read the card and try to resume without opening another tool.

Do not remind.

Observe only.

## Success

- 10 or more testers
- 3 successful re-entries

Successful re-entry means:

- user returns after time away
- user reads the card
- user understands where to resume in under 30 seconds
- user does not open another tool

## Stop Conditions

- people confused
- pressure increased
- board became another thing to maintain
