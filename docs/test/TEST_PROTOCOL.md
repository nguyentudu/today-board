# Test Protocol

## Goal

Observe return behavior.

## Cohort

10 users maximum.

## Duration

3-7 days.

## Instructions

1. Open board.
2. Create 1-3 cards.
3. Write short notes.
4. Close.
5. Return naturally.

Do not remind.

Observe only.

## Success

- 10 or more testers
- 3 or more returned
- 3 or more reported lighter

## Stop Conditions

- people confused
- pressure increased
- board became another thing to maintain
