# Re-entry Protocol

## Goal

Measure Time To Re-entry.

## Process

1. Create 1-3 cards.
2. Fill:
   - Why Still Open
   - If You Return
3. Leave for at least 48 hours.
4. Return naturally.
5. Attempt to resume context.

Measure manually.

## Success Gate

3 successful re-entries.

Successful re-entry means:

- user returns after at least 48 hours
- user recovers context in under 30 seconds
- user can continue without opening another tool

Allowed:

Not allowed:

- PMF
- Retention proven
- Commercial validation
- Product-market fit
